# Proyek Akhir 2 - SISTEM PEMBERIAN PAKAN DAN MONITORING KUALITAS AIR PETERNAKAN LELE PORTABLE, MENGGUNAKAN ENERGI TERBAHARUKAN (PANEL SURYA)

Repositori ini berisi kumpulan kode, program, dan skematik yang digunakan dalam Proyek Akhir 2. Berdasarkan nama folder dan file, proyek ini tampaknya merupakan sebuah sistem otomatisasi yang mengintegrasikan berbagai sensor dan aktuator.

## Deskripsi Proyek

Proyek ini adalah sistem otomatisasi (seperti *automatic feeder*) yang menggunakan mikrokontroler. Sistem ini mencakup fungsionalitas untuk pemberian pakan otomatis, kontrol perangkat (Relay Blower), pemantauan waktu (RTC), dan pembacaan sensor (Sensor pH).

## Komponen & Teknologi

Proyek ini tampaknya menggunakan komponen dan teknologi berikut:

* **Mikrokontroler** (Misalnya: Arduino Uno/Nano/Mega)
* **Bahasa Pemrograman**: C/C++ (Arduino)
* **Sensor**:
    * Sensor pH
    * Real-Time Clock (RTC) (Misal: DS3231)
* **Aktuator**:
    * Motor Servo
    * Modul Relay
    * Blower / Pompa
* **Output**:
    * Layar LCD (Misal: 16x2 I2C)
